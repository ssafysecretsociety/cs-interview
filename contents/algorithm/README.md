## Algorithm (알고리즘)
#### 알고리즘 기본
- 시간복잡도와 공간복잡도
- 완전 탐색 알고리즘 (Brute Force)
  - DFS와 BFS
  - 순열, 조합, 부분집합
- 백트래킹 (Backtracking)
- 분할 정복법 (Divide and Conquer)
- 탐욕 알고리즘 (Greedy)
- 동적 계획법 (Dynamic Programming)

#### 알고리즘 응용
- 정렬 알고리즘
- 그래프
  - 최단 경로 알고리즘
  - Union Find & Kruskal
- 두 포인터 (two-pointer)
- 문자열 처리 알고리즘
  - KMP 알고리즘
